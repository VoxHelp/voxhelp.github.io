Класс vox_list


Класс-реализация списка, может быть очень полезен



Функции



new


Создаёт экземпляр vox_list


Параметры


size - Размер списка. Если ничего не передать, то автоматически будет 0


Пример использования:

           
list = vox_list:new(6)
print(list:size())
list = vox_list:new()
print(list:size())

-- Вывод:
-- 6
-- 0



fromElement


Создаёт экземпляр vox_list с одним элементом


Параметры


element - Элемент, который добавится в список


Пример использования:

           
list = vox_list:fromElement("Perfect String")

print(list:size())
print(list:get(0))

-- Вывод:
-- 1
-- Perfect String



toArray


Конвертирует список в массив


Пример использования:

           
list = vox_list:fromElement("Perfect String")

array = list:toArray()

print(#array)
print(array[0])

-- Вывод:
-- 1
-- Perfect String



print


Выводит данные списка в терминал


Пример использования:

           
list = vox_list:fromElement("Perfect String")

list:print()

-- Вывод:
-- Size: 1
-- Iterations: 0
-- Perfect String



iterations


Возвращает число итераций для цикла for


Пример использования:

           
list = vox_list:new()

list:add("Perfect String")
list:add("Beautiful String")

iterations = list:iterations()

print(iterations)

for i = 0, iterations do
    print(list:get(i))
end

-- Вывод:
-- 1
-- Perfect String
-- Beautiful String



size


Возвращает число элементов в списке


Пример использования:

           
list = vox_list:new()

list:add("Perfect String")
list:add("Beautiful String")

print(list:size())

-- Вывод:
-- 2



addList


Добавляет все элементы из указанного списка в список


Параметры


list - Список с элементами


Пример использования:

           
stringsList = vox_list:new()

stringsList:add("Perfect String")
stringsList:add("Beautiful String")

list = vox_list:new()

list:addList(stringsList)

list:print()

-- Вывод:
-- Size: 2
-- Iterations: 1
-- Perfect String
-- Beautiful String



add


Добавляет элемент в список


Параметры


element - Элемент, который добавится в список


Пример использования:

           
list = vox_list:new()

list:add("Perfect String")

print(list:get(0))

-- Вывод:
-- Perfect String



set


Устанавливает на указанный индекс указанный элемент. Также если index больше чем размер списка, то список автоматически расширится до index


Параметры


element - Элемент, который добавится в список

index - Позиция на которую вставится элемент


Пример использования:

           
list = vox_list:new()

print(list:size())

list:set("Perfect String", 4)

print(list:size())
print(list:get(4))

-- Вывод:
-- 0
-- 4
-- Perfect String



get


Возвращает элемент с определённой позиции


Параметры


index - Позиция с которой нужно получить элемент


Пример использования:

           
list = vox_list:fromElement("Perfect String")

print(list:get(0))

-- Вывод:
-- Perfect String



clone


Клонирует список


Пример использования:

           
list = vox_list:new()

list:add("Perfect String")
list:add("Beautiful String")

listClone = list:clone()

print(listClone:size())
print(listClone:get(0))
print(listClone:get(1))

-- Вывод:
-- 2
-- Perfect String
-- Beautiful String



contains


Возвращает true, если указанный элемент содержится в списке


Параметры


element - Элемент, который нужно проверить на существование в списке


Пример использования:

           
list = vox_list:new()

list:add("Perfect String")

print(list:contains("Perfect String"))
print(list:contains("Beautiful String"))

-- Вывод:
-- true
-- false



indexOf


Возвращает позицию элемента в списке, возвращает -1 если элемента нет в списке


Параметры


element - Элемент, позицию которого нужно получить


Пример использования:

           
list = vox_list:new()

list:add("Beautiful String")

print(list:indexOf("Perfect String"))
print(list:indexOf("Beautiful String"))

-- Вывод:
-- -1
-- 0



remove


Удаляет указанный элемент из списка. Если удаление произведено успешно возвращает true, иначе false


Параметры


element - Элемент, который нужно удалить из списка


Пример использования:

           
list = vox_list:new()

list:add("Perfect String")

print(list:indexOf("Perfect String"))

list:remove("Perfect String")

print(list:indexOf("Perfect String"))

-- Вывод:
-- 0
-- -1



removeAt


Удаляет элемент на указанной позиции. Если удаление произведено успешно возвращает true, иначе false


Параметры


index - Позиция элемента, который нужно удалить


Пример использования:

           
list = vox_list:new()

list:add("Perfect String")

print(list:indexOf("Perfect String"))

list:removeAt(0)

print(list:indexOf("Perfect String"))

-- Вывод:
-- 0
-- -1
