Библиотека vox_arrays


Содержит утилиты для работы с массивами



Функции



createArrayFromElement


Создаёт массив из одного элемента


Параметры


element - Элемент из которого будет создан массив


Пример использования:

           
array = vox_arrays.createArrayFromElement("Perfect String")

print(#array)
print(array[0])

-- Вывод:
-- 1
-- Perfect String



array_contains


Возвращает true если массив содержит указанный элемент


Параметры


array - Массив

element - Элемент, который необходимо найти


Пример использования:

           
array = { "Perfect String" }

print(vox_arrays.contains(array, "Perfect String"))

-- Вывод:
-- true



toList


Конвертирует массив в список и возвращает экземпляр vox_list


Параметры


array - Массив


Пример использования:

           
array = { "Perfect String", "Beautiful String" }

list = vox_arrays.toList(array)

print(list:size())
print(list:get(0))
print(list:get(1))

-- Вывод:
-- 2
-- Perfect String
-- Beautiful String
