Класс vox_delegate


Класс, который позволяет хранить функцию и транспортировать её как делегат



Функции



new


Создаёт экземпляр vox_delegate


Параметры


entry - Таблица, в которой ref это функция


Пример использования:

           
function printString(str)
    print(str)
end

delegate = vox_delegate:new({ ref = printString })

delegate:invoke("Perfect String")

-- Вывод:
-- Perfect String



invoke


Вызывает функцию делегата


Параметры


params - Параметры, которые передадутся функции (неограниченное количество)


Пример использования:

           
function printString(str)
    print(str)
end
