Класс vox_container


Класс, который позволяет хранить и получать какое то значение



Функции



new


Создаёт экземпляр vox_container


Параметры


object - Обьект, который содержит контейнер


Пример использования:

           
container = vox_container:new("Perfect String")

print(container:get())

-- Вывод:
-- Perfect String



get


Возвращает обьект, который содержит контейнер


Пример использования:

           
container = vox_container:new("Perfect String")

print(container:get())

-- Вывод:
-- Perfect String



set


Изменяет содержимое контейнера


Параметры


object - Обьект, который будет содержать контейнер


Пример использования:

           
container = vox_container:new("Perfect String")

print(container:get())

container:set("Beautiful String")

print(container:get())

-- Вывод:
-- Perfect String
-- Beautiful String
