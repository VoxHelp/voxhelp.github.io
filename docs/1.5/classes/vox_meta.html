Класс vox_meta


Класс позволяющий хранить данные по координатам


Примечание: Можно использовать глобальный экземпляр просто обращаясь к vox_meta



Функции



new


Создаёт экземпляр vox_meta


Параметры


varsTable - Таблица, которую будет использовать экземпляр меты. Если ничего не передать, то создастся новая таблица


Пример использования:

           
varsTable = { x = { y = { z = { isChecked = false } } } }
meta = vox_meta:new(varsTable)
emptyMeta = vox_meta:new()



setVar


Устанавливает значение указанной переменной по координатам x, y, z


Параметры


x - Координата X

y - Координата Y

z - Координата Z

varName - Название переменной

value - Значение


Пример использования:

           
meta = vox_meta:new()

meta:setVar(0, 0, 0, "isChecked", false)



getVar


Возвращает значение указанной переменной по координатам x, y, z. Если переменная не установелна, возвращает nil


Параметры


x - Координата X

y - Координата Y

z - Координата Z

varName - Название переменной


Пример использования:

           
meta = vox_meta:new()

meta:setVar(0, 0, 0, "isChecked", false)

print(meta:getVar(0, 0, 0, "isChecked"))

-- Вывод:
-- false
