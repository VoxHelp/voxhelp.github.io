Библиотека vox_tables


Содержит утилиты для работы с таблицами



Функции



add


Добавляет значение к таблице


Параметры


tableVal - Экземпляр таблицы

value - Значение


Пример использования:

           
myTable = { }
vox_tables.add(myTable, "Perfect String")
print(myTable[0])

-- Вывод:
-- Perfect String



insert


Заменяет значение в таблице на указанной позиции


Параметры


tableVal - Экземпляр таблицы

index - Позиция

newValue - Новое значение


Пример использования:

           
myTable = { }
vox_tables.insert(myTable, 6, "Perfect String")
print(myTable[6])

-- Вывод:
-- Perfect String
